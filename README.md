# IPSec
IP Security tools. Simple gtk VPN menu.
# Requirements:
- OpenVPN
- zenity
# optional: 
- MACChanger
- AnonSurf

# Tested on:
- ParrotOS
- Kali

# *Instructions 
  cd into IPSec directory 
  
  sudo chmod +x IPSec
  
  sudo chmod +x IPSecAS
  
  sudo chmod +x IPSecC
  
  sudo chmod +x IPSecOVPNs
  
  Run in terminal
  
  ./IPSec
  
  Follow gtk prompt
